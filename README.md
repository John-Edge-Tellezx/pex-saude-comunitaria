# pex-saude-comunitaria
Pequeno portal web para registro e consulta de atendimentos comunitários em um posto de saúde do bairro. Permite cadastrar atendimentos, listar e filtrar registros e exportar dados para compor o relatório do PEX. Sistema para atuação local para demonstrar aplicação prática.

https://john-edge-tellezx.github.io/pax-saude-comunitaria/
